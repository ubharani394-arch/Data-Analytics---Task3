# ApexPlanet Data Analytics Project

## Project Description
This project demonstrates basic data analysis using Python, Pandas, and Matplotlib. The program loads an Excel dataset, explores the data, checks for missing values, converts date columns into datetime format, creates a new month column, and calculates the total sales.

## Features
- Load an Excel dataset
- Display the first five records
- View dataset information
- Generate summary statistics
- Check for missing values
- Convert `Order_Date` to datetime format
- Extract the month from the order date
- Calculate total sales

## Technologies Used
- Python
- Pandas
- Matplotlib

## Code Workflow

1. Import the required libraries.
2. Load the Excel dataset using Pandas.
3. Display the first five rows of the dataset.
4. Display dataset information.
5. Display statistical summary.
6. Check for missing values.
7. Convert the `Order_Date` column to datetime format.
8. Create a new `Month` column from `Order_Date`.
9. Calculate and display the total sales.
